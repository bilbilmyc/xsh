# XSH Roadmap

## 已完成：MVP + V2 核心能力

- Rust workspace、Tauri 2、React/xterm.js 桌面架构
- SQLite 会话目录与 XSH AES-256-GCM 自建凭据库
- 密码、OpenSSH Key、SSH Agent、keyboard-interactive 认证
- TOFU Known Hosts、SHA-256 指纹和 Host Key 变化阻断
- `~/.ssh/config` 导入、连接诊断、错误分类、脱敏诊断摘要、ProxyJump 和端口转发
- 多标签常驻 SSH、标签锁定与关闭保护、自定义标签颜色、左右/上下分屏、启动恢复、命名工作区、密码加密工作区导入/导出和命令广播
- 自动重连、Keepalive、日志、搜索、字体缩放和可配置剪贴板行为
- 命令中心、命令历史、最近连接、快捷命令分组/搜索/快捷键
- 快捷命令敏感内容拒绝保存与旧数据安全清理
- SFTP 文件/目录递归传输、非空目录递归删除、多选、批量操作、拖拽上传、远程编辑、真实并发队列、冲突策略、取消、重试和下载断点续传
- 会话目录创建、重命名、删除、单项/批量拖拽整理，会话批量移动/收藏/删除、环境/自动重连/标签批量编辑，以及同名或相同连接目标保存前提示
- macOS 免费个人版打包、Windows 构建脚本和跨平台 CI

## 下一阶段：实机质量与发布工程

1. Windows 10/11 x64 实机安装、WebView2、字体、剪贴板、SSH/SFTP 回归测试
2. macOS 长时间多标签稳定性、睡眠唤醒、网络切换和大文件传输压力测试
3. SFTP 可选连接限速、上传断点续传和并发偏好
4. 工作区与会话配置的跨机器联合迁移向导（合并密码输入、预览、冲突策略和凭据恢复）
5. 会话批量终端配置编辑和导入冲突可视化
6. 完整键盘焦点移动、屏幕阅读器回归和高 DPI/多显示器视觉验收
7. 可选的本地诊断日志导出与性能可观测性
8. 如未来需要公开分发，再加入 Apple 公证与 Windows 代码签名

## 明确不做

SSH1、Telnet、Rlogin、RDP、X11 forwarding、ZMODEM、Kerberos、PKCS#11、云同步和团队共享凭据暂不进入当前版本。
